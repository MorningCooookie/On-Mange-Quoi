Implementation Notes

CSS Variables

All colors, fonts, and spacing are defined as CSS variables in :root. This enables easy dark mode switching and theme consistency.

Dark Mode

Dark mode uses :root[data-theme="dark"] selector with localStorage persistence. System preference is checked on load via prefers-color-scheme.

Typography Stack

Fonts loaded from Google Fonts CDN: Fraunces (display), Work Sans (body), IBM Plex Mono (data).

Responsive Design

Mobile-first approach. Breakpoint at 768px switches to multi-column layouts. All grids use repeat(auto-fit, minmax()) for fluid responsiveness.

Button Styling

Primary buttons: background: var(--primary); color: white; padding: 0.75rem 1.5rem; border-radius: 6px; with hover transform and shadow.

Transitions

All interactive elements use transition: all 200ms ease-in-out; for smooth, GPU-accelerated animations. Avoid animating width/height; use transform instead.

Accessibility

Color contrast tested against WCAG AA minimum (4.5:1 for text). Focus states use border + subtle shadow rather than outline.
